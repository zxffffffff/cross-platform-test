# cross-platform-test
横向对比测试跨平台框架的性能，包括 Qt、Electron、Flutter

压测
- 统一尺寸 1920 x 1080，列表控件 40 x 20（直接影响UI绘制的耗时）
- 单次请求：点击-网络请求-网络解析-UI重绘（刷新所有控件）
- 持续请求：定时 10ms 网络请求-网络解析-UI重绘（性能瓶颈会导致Timer延迟）
- 高频绘制：鼠标输入-UI重绘（查看鼠标延迟）
- 主观评价： 技术栈，学习成本，开发周期、开发复杂性、生态
